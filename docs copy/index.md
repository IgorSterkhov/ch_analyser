# ClickHouse Analyser - Documentation

Инструмент для анализа хранилища ClickHouse: размеры таблиц, детализация по колонкам, история запросов и информация о дисках. Работает через веб-интерфейс (NiceGUI) и десктопный интерфейс (Tkinter).

## Для пользователей

- [Руководство пользователя](user-guide.md) - как работать с приложением: подключение, анализ таблиц, история запросов
- [Руководство администратора](admin-guide.md) - управление подключениями, пользователями и настройками
