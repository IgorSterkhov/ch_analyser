  Возможные дальнейшие доработки для мониторинга ClickHouse

  Дисковое пространство и хранение:
  - Размер каждой базы данных отдельно (system.parts GROUP BY database)
  - Топ-N самых быстро растущих таблиц (сравнение snapshot'ов по времени)
  - Соотношение сжатия данных (compressed vs uncompressed из system.parts)
  - Мониторинг отдельных партиций и их возраста

  Производительность и нагрузка:
  - Текущие активные запросы (system.processes) — что выполняется прямо сейчас
  - Медленные запросы — топ по query_duration_ms из system.query_log
  - Статистика по пользователям — кто больше нагружает сервер
  - Метрики сервера (system.metrics, system.events) — memory usage, merge count, connections  

  Здоровье системы:
  - Статус репликации (system.replicas) — отставание реплик
  - Мониторинг мержей (system.merges) — текущие и очередь
  - Ошибки и warnings из system.errors
  - Uptime и версия сервера

  Визуализация:
  - Графики роста таблиц во времени
  - Dashboard с автообновлением
  - Экспорт отчётов (CSV/PDF)
